# [雀魂Plus - 雀魂麻将第三方桌面客户端](https://github.com/iamapig120/majsoul-plus-client)

- 除首次加载较慢，其余均可高速加载
- 轻量级，仅依赖 express 、 electron 以及 AdmZip
- 桌面端跨平台
- 本地文件不加密
- 支持魔改模组和代码执行

## 如何使用

1. 克隆或下载本仓库
2. npm install
3. npm start

在客户端中玩的开心！
